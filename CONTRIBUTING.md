# Contributing

Thank you for your interest in contributing! 🎉

This repository is a community-driven educational resource designed to help SOC Analysts, Blue Teamers, Threat Hunters, Incident Responders, DFIR professionals, Security Engineers, and cybersecurity students understand Windows Security Event IDs.

Contributions of all sizes are welcome.

---

## Ways to Contribute

You can contribute by:

- Adding missing Windows Event IDs
- Improving existing explanations
- Fixing inaccuracies or typos
- Adding investigation tips
- Improving MITRE ATT&CK mappings
- Adding detection ideas
- Improving PowerShell, Sysmon, or Windows Defender documentation
- Enhancing formatting and readability
- Suggesting new sections or features

---

## Before You Submit

Please ensure that:

- Information is technically accurate.
- Content is written in clear Markdown.
- Event descriptions are concise and easy to understand.
- References are from reputable sources whenever possible.
- Existing formatting and style are maintained.

---

## How to Contribute

1. Fork this repository.
2. Create a new branch.
3. Make your changes.
4. Commit your changes with a meaningful commit message.
5. Push your branch.
6. Open a Pull Request.

Please describe:

- What was changed
- Why it was changed
- Any useful references (if applicable)

---

## Reporting Issues

Found an error or missing information?

Please open a GitHub Issue and include:

- The affected file
- The Event ID (if applicable)
- A description of the issue
- Suggested improvements (optional)

---

## Code of Conduct

Please be respectful and constructive.

This project aims to provide a welcoming environment for everyone interested in cybersecurity.

---

## Thank You

Every contribution—whether it's fixing a typo, improving documentation, or adding new Event IDs—helps make this resource more useful for the cybersecurity community.

Happy hunting! 🛡️
